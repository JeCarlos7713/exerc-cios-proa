Quem somos?

A Outset é um projeto criado em 2021 que visa iniciar o futuro profissional dos nossos alunos através da tecnologia. Trazemos como cultura o ensino gratuito, oferecendo cursos, palestras e materiais de apoio para ajudar você a dar um start na sua carreira. Foi dada a largada, você vem?

O projeto

Se você leu até aqui, significa que quer dar esse start em sua carreira profissional. Mas, como o projeto funciona? Oferecemos um curso de Desenvolvimento Web onde os nossos alunos aprendem desde abrir uma IDE de desenvolvimento e fazer o seu primeiro código, até criar um site com um sistema integrado. No curso você irá adquirir uma boa base de Front-end e Back-end, se tornando um Fullstack de sucesso. E ai, se interessou mesmo? Entre em contato com a gente e saiba mais.
